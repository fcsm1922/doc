 ## Работа с архивами

 ### ZIP - архивы

1. Установка пакета `unzip`

  ```
  sudo pacman -S unzip
  ```
2. Извлечь данные из ZIP-файла
  ```
  unzip filename.zip
  ```
> Файл будет извлечен в тот же каталог, где находится ZIP-файл
3. Распаковать ZIP-файл в другой каталог 
  ```
  unzip filename.zip -d /path/of/directory
  ```
